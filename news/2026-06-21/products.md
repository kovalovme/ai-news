## Grok 4.3 on Amazon Bedrock <a id="grok-43-bedrock"></a>

**Source:** [AWS What's New](https://aws.amazon.com/about-aws/whats-new/2026/06/grok-amazon-bedrock/) · [xAI](https://x.ai/news/grok-amazon-bedrock) · **Type:** launch · **Time (UTC):** Jun 15

xAI's Grok 4.3 is now available as a first-party model on Amazon Bedrock — the first xAI model on the platform. The model offers a 1-million-token context window, native video input, configurable reasoning effort (none / low / medium / high), tool calling, structured output, and response streaming. It runs on Mantle, a new Bedrock inference engine built for price-performance efficiency.

Pricing is **$1.25 per million input tokens** and **$2.50 per million output tokens**, positioning Grok 4.3 as one of the most affordable frontier reasoning models on Bedrock.

**Why it matters:** Bedrock availability removes the need for enterprises with existing AWS infrastructure to add a separate xAI API integration or custom authentication layer. The configurable reasoning depth is particularly useful for agentic workloads where lightweight calls dominate but occasional high-reasoning steps are needed — allowing per-call cost tuning without switching models.

| Spec | Value |
|------|-------|
| Context window | 1M tokens |
| Input price | $1.25 / M tokens |
| Output price | $2.50 / M tokens |
| Reasoning modes | none, low, medium, high |
| Native video input | Yes |
| Inference engine | Mantle (Bedrock-native) |

## OpenAI Partner Network <a id="openai-partner-network"></a>

**Source:** [OpenAI](https://openai.com/index/introducing-openai-partner-network/) · **Type:** launch · **Time (UTC):** Jun 14-15

OpenAI launched its first formal partner ecosystem program with $150 million committed across three tiers (Select, Advanced, Elite), differentiated by sales volume, technical capability, and deployment track record. The target is 300,000 certified consultants trained by end of 2026. Founding partners include Accenture, Bain, BCG, Eliza, McKinsey, and PwC.

A Forward Deployed Experts initiative co-locates qualified partner practitioners with OpenAI's own engineering teams on complex enterprise deployments. OpenAI framed the rationale explicitly: "The limiting factor for seeing value from AI in the enterprise is no longer model capabilities — it's how organizations repeatably identify use cases, redesign workflows, integrate systems, and drive adoption at scale."

**Why it matters:** OpenAI is building the same certified-reseller infrastructure that Anthropic stood up with its Claude Partner Network on June 3, 2026. The $150M commitment and 300K consultant target are substantially larger numbers, but OpenAI is launching three weeks later. For engineers at system integrators, the tiered certification structure signals where partner margin and co-sell pipeline will concentrate in the second half of 2026.
