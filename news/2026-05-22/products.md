# Products — 2026-05-22

## Google Search Information Agents and Intelligent Search Box <a id="google-search-information-agents"></a>

**Source:** [Google Blog](https://blog.google/products-and-platforms/products/search/search-io-2026/) · [TechCrunch](https://techcrunch.com/2026/05/19/google-search-as-you-know-it-is-over/) · [The Next Web](https://thenextweb.com/news/google-search-ai-overhaul-information-agents-io-2026) · **Type:** launch · **Time (UTC):** rolling out this week

Google announced two interconnected changes to Search at I/O 2026 (May 19) that are now beginning global rollout:

**Intelligent Search Box** (rolling out globally this week): The classic single-line input is replaced with a dynamically expanding panel that accommodates multi-sentence queries, accepts image, video, file, and browser-tab inputs alongside text, and surfaces AI-powered intent suggestions that go beyond autocomplete. The box is the same entry point across AI Mode and traditional Search.

**Information Agents** (launching summer 2026, U.S. AI Pro and Ultra subscribers first): Persistent background agents that monitor the web on behalf of the user and deliver synthesized push updates when conditions match a stated interest. Example use cases: track apartment listings against a set of specific requirements; monitor a stock for news matching a custom thesis; follow a regulatory topic across agencies. Agents reason across web content, real-time finance/sports/shopping feeds, and social posts, then send an "intelligent, synthesized update with the ability to take action." Gemini 3.5 Flash is now the default model powering AI Mode for all global users.

These features are distinct from Gemini Spark (covered in [2026-05-20/products.md](../2026-05-20/products.md#gemini-spark)), which is a personal agent operating on tasks delegated by the user. Information agents operate proactively on standing interests without per-session delegation.

**Why it matters:** Search information agents represent Google's entry into the ambient monitoring category currently occupied by custom alert tools, Bloomberg Terminal watchlists, and RSS-based notification setups. Embedding this capability at the default Search entry point — with a billion-scale user base — gives Google instant distribution that purpose-built agent products cannot match. For the broader market, a world where Google's search box becomes a persistent agent changes the unit economics of information intermediaries, SEO-driven publishing, and specialized data-monitoring SaaS products.
