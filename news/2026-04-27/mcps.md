# MCPs — 2026-04-27

## MCP governance update: new Lead Maintainer and Core Maintainer appointed <a id="mcp-governance-update"></a>

**Source:** [Model Context Protocol Blog](https://blog.modelcontextprotocol.io/posts/2026-04-08-maintainer-update/) · **Type:** update · **Time (UTC):** Apr 8 —

The Model Context Protocol project announced two governance changes. Den Delimarsky — Anthropic MTS who has worked across the MCP ecosystem since launch — stepped up from Core Maintainer to Lead Maintainer. Clare Liguori joined as a new Core Maintainer, bringing developer-tooling and agent-runtime experience to the protocol's design work. The blog post cites accelerating enterprise adoption and increased specification complexity as the drivers behind formalizing the maintainer structure.

**Why it matters:** MCP's 2026 roadmap priorities (Streamable HTTP transport, Tasks primitive, enterprise SSO/audit) require faster and more authoritative spec decisions. Naming a Lead Maintainer signals that the protocol is moving from an experimental Anthropic project toward a more independently governed standard, which matters for vendors building on it.
