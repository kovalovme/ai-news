# Products — 2026-05-29

## Anthropic Signals Mythos-Class General Availability "in Coming Weeks" <a id="mythos-public-timeline"></a>

**Source:** [Axios](https://www.axios.com/2026/05/28/anthropic-opus-release-mythos) · **Type:** update · **Time (UTC):** May 28

Alongside the Opus 4.8 release, Anthropic stated that Mythos-class models — currently limited to the Project Glasswing security partner program — will become available to all customers "in the coming weeks," contingent on completing enhanced safety guardrails. No specific date, pricing tier, or access model was announced. Opus 4.8 scores comparably to Mythos on prosocial and user-autonomy benchmarks; Mythos remains superior on security vulnerability discovery tasks.

**Why it matters:** This is the first stated public timeline for Mythos general availability since the Glasswing program launched in April 2026. A publicly accessible Mythos-class model would represent the highest capability threshold ever offered for automated code security analysis outside of controlled partner programs.
