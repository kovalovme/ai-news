# Products — 2026-06-11

## OpenAI Models and Codex via Oracle Cloud Commitments <a id="openai-oracle-cloud"></a>

**Source:** [OpenAI blog](https://openai.com/index/openai-on-oracle-cloud/) · **Type:** launch · **Time (UTC):** 2026-06-11

OpenAI announced that enterprise customers with existing Oracle Cloud commitments — tracked through Oracle's Universal Credits (UCM) program — can now apply those credits directly toward OpenAI API usage, covering frontier models and Codex. Customers access models through Oracle Cloud Infrastructure (OCI) without establishing a separate OpenAI billing relationship; credits are consumed from their existing Oracle agreement. Availability begins within the coming weeks. The announcement is part of the larger Stargate infrastructure partnership: Oracle is building Stargate data center capacity at five new sites, with the first NVIDIA GB200 racks delivered to the Abilene, Texas flagship campus in June.

**Why it matters:** For the large portion of enterprise IT spend already locked into Oracle Cloud commitments, this removes a procurement barrier to OpenAI adoption — no separate vendor contract, no new budget approval cycle needed. It also positions Oracle as an AI distribution channel alongside the major hyperscaler marketplaces (AWS, Azure, Google Cloud).
