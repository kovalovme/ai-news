# Models — 2026-06-07

## Gemini 3.5 Pro: 2M context and Deep Think mode confirmed <a id="gemini-3-5-pro"></a>

**Source:** [TechTimes](https://www.techtimes.com/articles/317919/20260606/google-gemini-35-pro-nears-june-launch-2-million-token-context-deep-think-reasoning.htm) · [WaveSpeed Blog](https://wavespeed.ai/blog/posts/gemini-3-5-pro-coming-next-month/) · **Type:** pre-launch · **Time (UTC):** Jun 6 reporting

Google has not released Gemini 3.5 Pro to general users as of June 7, but the model's specifications are now confirmed: a 2-million-token context window, a "Deep Think" reasoning mode designed to trade throughput for hard-problem accuracy, and expected pricing of approximately $15 input / $60 output per million tokens — roughly 10× Flash's rates. Sundar Pichai stated at Google I/O 2026 (May 19) to "give us until next month," and the model remains in internal use and limited enterprise preview with broad availability still pending.

**Why it matters:** At 2M tokens, Pro would extend Gemini 3.5 Flash's 1M-token context limit and position Google against Claude Opus 4.8's frontier long-context work. Independent benchmarks remain unavailable, so published specifications should be treated as targets rather than verified performance until GA.

| Specification | Value |
|---|---|
| Context window | 2,000,000 tokens |
| Reasoning mode | Deep Think (high-compute) |
| Expected input price | ~$15 / 1M tokens |
| Expected output price | ~$60 / 1M tokens |
| Status (Jun 7) | Internal / limited preview only |
