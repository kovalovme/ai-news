## Project Glasswing: 10,000+ Critical Bugs Found, Partners Can Now Share Findings <a id="glasswing-update"></a>

**Source:** [Benzinga / Anthropic](https://www.benzinga.com/markets/private-markets/26/05/52759147/anthropics-project-glasswing-finds-more-than-10000-critical-bugs-expands-to-additional-partners) · **Type:** update · **Time (UTC):** 2026-05-22 ~15:00

Anthropic published its first substantive progress report on Project Glasswing, the controlled-access program that gives vetted partners early access to its Mythos Preview security model for defensive vulnerability research. Key results since the program's April 2026 launch: Mythos Preview identified an estimated 6,202 high- or critical-severity vulnerabilities across 1,000+ open-source projects; combined partner findings across all tools exceed 10,000 such vulnerabilities. Cloudflare reported roughly 2,000 bugs including 400 high-or-critical, with fewer false positives than their conventional testing baseline. Mozilla fixed 271 vulnerabilities in Firefox 150 using the system. Anthropic also disclosed that the system detected and blocked a fraudulent $1.5M wire transfer in a live deployment. Two policy changes accompany the update: Glasswing partners may now publish findings, tools, or code they developed using Mythos (reversing the original restriction), and Anthropic plans to onboard additional partners including U.S. and allied government agencies. The company emphasized that bug discovery is outpacing the industry's capacity to patch and deploy fixes.

**Why it matters:** The permission change enabling partners to share findings opens the ecosystem for peer review and tooling interoperability — the closed-garden approach had been a friction point for security researchers. The rate of discovery (10,000+ critical bugs in roughly six weeks) is a concrete data point on what a frontier model can do in an adversarial code review context.
